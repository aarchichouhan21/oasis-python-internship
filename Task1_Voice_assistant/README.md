Voice Assistant(Python)
This project is a simple voice assistant developed using python as part of the Oasis Infobyte Internship.
Features
- Speech Recognition
- Voice response
- Basic commands like greeting and time
- Uses microphone input

Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- Pyaudio

How To Run 
- Install required libaries:
  pip install speechrecognition pyttsx3 pyaudio
