Advanced BMI Calculator
A Python GUI application that calculates Body Mass Index (BMI), provides health advice, and tracks BMI progress over time.
1.Features
- GUI interface using Tkinter
- BMI calculation
- BMI category classification
- Health advice
- BMI history tracking
- BMI progress graph visualization
2.Technologies Used
- Python
- Tkinter
- Matplotlib
3.Installation
Install dependencies:
- pip install -r requirements.txt
Run the program:
- python BMI_Calculator.py
